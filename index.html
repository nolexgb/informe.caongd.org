Esta versión rehace profesionalmente la visualización del informe CAONGD 2023 e integra un motor de datos para generar automáticamente el informe anual, explorar dimensiones y visualizar resultados de forma dinámica y personalizable.

Resumen 2023

Explorador dinámico

El usuario puede integrar variables y cambiar la lectura.

Selecciona una vista, una dimensión y una métrica. El sistema recompone automáticamente tarjetas, rankings y tablas. Esta lógica es la base para que cada nuevo año se publique con la misma estructura sin rehacer la web.

Sistema de generación automática

Un solo archivo de datos, múltiples salidas visuales.

La herramienta propuesta permite integrar diversas variables, cargar series anuales y regenerar automáticamente un informe interactivo con filtros, rankings, comparativas y lectura territorial.

Flujo automático
Recoger datos → validar → transformar → visualizar → publicar

El modelo separa contenido y presentación. Eso permite actualizar el informe con nuevos datos, añadir variables y mantener consistente toda la experiencia interactiva.

Rankings y focos

Lecturas ejecutivas de alto valor.

Además de la navegación libre, el sistema destaca automáticamente territorios, áreas y entidades con mayor volumen relativo para facilitar lectura rápida a equipos técnicos, financiadores y comunicación.

Base social e implantación

El informe también mide legitimidad social.

Se incorporan indicadores sobre personas socias, voluntariado, personal contratado, personal expatriado y composición de órganos de gobierno. Esta capa puede seguir creciendo con tramos de edad y evolución anual.

Arquitectura recomendada de la herramienta

Frontend estático premium + archivo JSON anual + posibilidad de conectar formularios o base de datos. El sistema puede crecer hacia paneles comparativos 2022/2023/2024, filtros múltiples, exportación de vistas y cuadros específicos para Andalucía, otros países, ODS, financiadoras, provincias y grupos de población.

Diseñado para que la CAONGD pueda publicar un nuevo año cambiando datos, no rehaciendo páginas.
